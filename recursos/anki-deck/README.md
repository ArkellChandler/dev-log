# 🎴 Anki Deck: Cyberpunk HUD Style
Este diretório contém os baralhos de estudo (Flashcards) e a folha de estilos personalizada com tema Cyberpunk para o Anki.

## 📂 Conteúdo
- sintaxe_redes_top.csv: Baralho focado em comandos de Redes, PowerShell e SQL.
- 	eoria_informatica_top.csv: Baralho focado em Modelo OSI, TCP/IP e Conceitos de Banco de Dados.
- nki_cyberpunk_config.txt: Guia de configuração de CSS e modelos de Cartão (Frente/Verso).

## 🚀 Como Utilizar

### 1. Importação dos Dados
No Anki, vá em **Arquivo -> Importar** e selecione um dos arquivos .csv.
- **Importante:** Configure o separador como **Ponto e Vírgula (Semicolon)**.

### 2. Aplicação do Estilo (Visual HUD)
Abra o arquivo nki_cyberpunk_config.txt e siga as instruções:
1. Copie o código do **Modelo da Frente** para a seção correspondente no Anki.
2. Copie o código do **Modelo do Verso**.
3. Copie todo o conteúdo de **Estilo (CSS)** para a aba "Styling".

## 🛠️ Campos Necessários
Para o funcionamento correto dos scripts de embaralhamento e visualização, certifique-se de que seu Tipo de Nota possua os seguintes campos:
- **Teoria:** Question, Correct Answer, Incorrect Answer 1, Incorrect Answer 2, descricao.
- **Sintaxe:** Frente, Verso, descricao.
