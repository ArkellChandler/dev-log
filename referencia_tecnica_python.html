📊 Tabela Técnica: Guia de Referência Python

Elemento Sintaxe Descrição Literal Cuidado / Boa Prática
Lista Mutável [item1, item2] Coleção ordenada que aceita duplicatas e alterações. Cuidado com o índice: 0 é o primeiro. Estoura erro se acessar índice que não existe.
Tupla Imutável (item1, item2) Coleção ordenada e travada. Mais rápida e segura. Não permite append ou pop. Use para dados que nunca mudam.
Dicionário Mutável {"chave": "valor"} Dados organizados por etiquetas únicas (chaves). Chaves devem ser únicas. .items() é necessário para ver chave e valor no loop.
Conjunto (Set) Mutável {item1, item2} Sacola de itens únicos. Sem ordem e sem duplicatas. Não tem índice (não existe set[0]). Apaga repetidos automaticamente.
Def (Função) def nome(): Cria uma máquina de ação reutilizável. Sempre use : no final. O código interno deve ter 4 espaços de recuo.
If / Else if condicao: O portão de decisão. Escolhe caminhos lógicos. Python é Case-Sensitive! "Oi" é diferente de "oi". Use == para comparar.
For (Loop) for item in lista: Percorre coleções, item por item. A variável temporária (item) só existe "dentro" do loop.
.append() lista.append(x) Adiciona um item ao FINAL de uma lista. Só funciona em Listas. Não use em Tuplas.
.pop() lista.pop(indice) Extrai e remove um item pela POSIÇÃO. Se vazio, remove o último. Devolve o item removido se quiser guardar.
.remove() lista.remove(nome) Apaga um item pelo NOME/VALOR literal. Gera ERRO se o nome não existir na lista. Verifique antes.
.add() set.add(x) Adiciona um item a um Conjunto. Exclusivo para Conjuntos (Sets).
del del lista[0] Comando de destruição por posição ou variável. É definitivo. Não devolve o item apagado. Cuidado ao usar.
