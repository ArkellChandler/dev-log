Guia de Referência: Integração SQL

Categoria Comando / Código Descrição
COMANDOS SQL (SQLite)
Criar Tabela CREATE TABLE alunos (id INTEGER PRIMARY KEY, nome TEXT, nota REAL); Cria a estrutura da tabela no banco de dados.
Inserir Dado INSERT INTO alunos (nome, nota) VALUES ('Arkell', 10.0); Adiciona um novo registro na tabela.
Consultar Tudo SELECT * FROM alunos; Mostra todos os registros da tabela.
Filtrar Dados SELECT * FROM alunos WHERE nota > 9.0; Busca registros que atendem a uma condição.
Atualizar Dado UPDATE alunos SET nota = 9.5 WHERE nome = 'João'; Altera informações de um registro existente.
Remover Dado DELETE FROM alunos WHERE nome = 'Maria'; Exclui um registro da tabela.
CONEXÃO PHP
Conectar $db = new SQLite3('estudos.db'); Inicia a conexão com o banco de dados via PHP.
Executar Query $resultado = $db->query("SELECT * FROM alunos"); Envia o comando SQL para o banco através do PHP.
Execução Terminal php.exe -d extension=sqlite3 leitura.php Comando para rodar o script PHP com a extensão do SQLite ligada.
CONEXÃO PYTHON
Importar import sqlite3 Importa a biblioteca nativa do Python para SQL.
Conectar conn = sqlite3.connect('estudos.db') Cria o canal de comunicação entre Python e o Banco.
Buscar Tudo cursor.fetchall() Coleta todos os dados retornados pela consulta SQL.
